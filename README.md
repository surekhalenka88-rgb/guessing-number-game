# 🎯 Guessing Number Game (Python)

## 📌 Project Description
This project is a simple Number Guessing Game developed using Python.

The program generates a random number between **1 and 10**, and the user keeps guessing until the correct number is entered.

## ✨ Features
- Random number generation
- User input handling
- Too High / Too Low hints
- Infinite attempts until correct guess
- Beginner-friendly Python project

## 🛠 Technologies Used
- Python
- Random Module

## 📂 Project Structure
```
guessing_number.py
README.md
```

## ▶️ How to Run

1. Install Python
2. Open terminal
3. Run:

```bash
python guessing_number.py
```

## 💻 Sample Output

```text
Enter Your Guess: 4
Too Low

Enter Your Guess: 9
Too High

Enter Your Guess: 7
Hey! Congrats, you guessed the number 🎉
```

## 🔄 Program Flow
1. Generate random number
2. User enters guess
3. Compare values
4. Show hint
5. Repeat until correct
